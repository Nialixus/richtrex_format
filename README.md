# RichTrex: Format

An extended package of `RichTrex` package which is used to encode `TextSpan` into `String`, and decode `String` into `TextSpan`.

## Primary Format

- Font Color ✅
- Bold ✅
- Font Size ✅
- Font Family ✅
- Custom ❌
- Italic ✅
- Underline ✅
- Strikethrough ✅
- Background Color ✅
- Image ✅
- Align ✅
- Ordered List ❌
- Unordered List ❌
- Block Quote ❌
- Link ❌

## Secondary Format

- Superscript ❌
- Subscript ❌
- Shadow ✅
- Height ✅
- Space ✅
- Overline ✅
- Table ❌
- Video ❌
